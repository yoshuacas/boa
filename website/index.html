How it works

Three steps. Your agent does the heavy lifting.

Install the skill

Add BOA to your coding agent. It takes 10 seconds and works across Claude Code, Kiro, VS Code, and Codex.

claude plugin install boa

Tell your agent what to build

Describe your app. BOA guides your agent through the right architecture — database schema, auth, APIs, storage — and deploys it to your AWS account.

"Build a social app with user profiles, posts, and image uploads"

Manage your backend

BOA includes a local dashboard — static HTML pages that connect to your AWS CLI — to visualize and manage your backend. No separate server needed.

open .boa/dashboard/index.html

The Stack

Opinionated. Serverless. Production-ready.

Database
Aurora DSQL
Serverless PostgreSQL. Scales to zero. Multi-AZ. SQL you already know.
Auth
Amazon Cognito
User sign-up, sign-in, MFA, social login. First 10,000 users free.
Compute
AWS Lambda
Node.js 20.x functions. Runs your business logic. 1M free requests/month.
API
API Gateway (REST)
Managed REST APIs with Cognito authorization built in.
Storage
Amazon S3
File uploads via presigned URLs. Never public. 5 GB free.
Hosting
AWS Amplify
Deploy your frontend. CI/CD from Git. Custom domains. Free tier included.
Infrastructure
SAM / CloudFormation
Repeatable deployments. Your agent writes the templates.
Dashboard
Local HTML + AWS CLI
Manage your backend locally. No servers. Pure HTML reading your account.
